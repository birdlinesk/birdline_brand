## 📤 CANVAS UPLOAD CHECKLIST

**Step 1: Prepare**
- [ ] Have all files from VISUAL_ASSETS/ ready
- [ ] Read CANVAS_UPLOAD_GUIDE.md
- [ ] Copy company blurb text (provided in guide)

**Step 2: Create System in Canvas**
- [ ] Go to canvas.dev/design-systems
- [ ] Click "Create new design system"
- [ ] Fill "Company name and blurb" field
- [ ] Paste company description

**Step 3: Upload Assets**
- [ ] Upload Birdline_Logo.svg
- [ ] Upload Birdline_Logo_Black.svg
- [ ] Upload Birdline_Logo_White.svg
- [ ] Upload Birdline_Canvas_Brand.png
- [ ] Upload listy.ai

**Step 4: Add Brand Philosophy**
- [ ] Scroll to "Any other notes?"
- [ ] Paste full brand philosophy text (from guide)
- [ ] Include color palette reference
- [ ] Include typography guidelines

**Step 5: Finalize**
- [ ] Review all entries
- [ ] Click "Create Design System"
- [ ] Your Canvas design system is ready!

## 🎯 BRAND PRINCIPLES

**✅ DO:**
- Use Neon colors for energy and clarity
- Keep typography minimal and purposeful
- Embrace geometric forms and reduction
- Think: Guide through complexity
- Deliver museum-quality craftsmanship

**❌ DON'T:**
- Use generic fonts (no Arial, Inter)
- Mix with flight/aviation metaphors
- Apply heavy shadows
- Overcomplicate the design
- Forget the toucan guides
